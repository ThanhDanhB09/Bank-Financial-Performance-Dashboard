# LPBank Financial Performance Dashboard (2019 - 2023)

## 📌 Project Overview
This project delivers an end-to-end Business Intelligence (BI) solution that provides interactive, data-driven visibility into LPBank's financial health. It automates financial reporting, processes complex datasets, and pinpoints key operational insights to assist strategic decision-making.

## 📂 Data Source
The raw dataset was sourced and consolidated from public financial statements available on the **Cophieu68** platform . It includes comprehensive balance sheets and income statements spanning from 2019 to 2023, extracted automatically using Excel Web Queries. https://www.cophieu68.vn/quote/financial_detail.php?id=lpb&amp;type=year

## 🛠 Tools & Technologies
- **Power BI Desktop & Service**
- **DAX (Data Analysis Expressions)**
- **Power Query (ETL Processes)**
- **Microsoft Excel (Web Query & Analytics)**

## 💡 Key Highlights
- **Commercial Banking Domain:** Visualized core banking metrics including Loan-to-Deposit Ratio (LDR Trend), net interest components, and Earnings Per Share (EPS).
- **Data Modeling & ETL:** Sourced and consolidated multi-year financial statements from public web data via Power Query; cleaned, unpivoted, and transformed complex datasets into an optimized star-schema model.
- **Advanced DAX Optimization:** Developed dynamic DAX measures for Year-over-Year (YoY) growth calculations (Asset +16.8%, Profit +23.5%), asset distribution, and granular cost breakdowns.

## 🚀 Project Outcomes
Delivered an end-to-end BI solution that automated financial reporting, cutting manual workloads. The dashboard enhanced visibility into the bank's financial health and enabled data-driven decisions by identifying key cost drivers (e.g., 63.03% Interest Expense) and core revenue segments.
